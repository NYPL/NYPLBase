Icon font

This is the current set of supported icons. We use a custom export of Icomoon.

Note to self: watch for width conflicts on elements with both a width class ("one-fifth") and an icon class or data-icon on the same element; icons have a width and height: auto setting.

How to use the icons

Class name

This is some text

				<p class="icon-cog">This is some text</p>

Data-icon attribute

This is also some text

				<p aria-hidden="true" data-icon="&#xf013;">This is also some text</p>

Mixin

Feel free to incorporate icons into your own styles, like this:

				.custom-zoom-out:before { @include nypl-icon("-"); }
				.custom-zoom-in:before { @include nypl-icon("+"); }

The logo

This is the logo (without the "New York Public Library" text, it's referred to as the "lion mark"), conveniently available as part of our icon font! Because it's an icon font, changing the color of the lion mark is as easy as applying a text color.

See? All I did was change the color of the text of the element with the icon. Now it's blue!

We also have a version of the lion mark in reverse. Here it is, no styling applied.

But what you see here is TOTALLY WRONG. Bad, wrong, no-no. Never use the lion mark with black/dark fills. The thing to remember is:
the fill is always lighter than the background!

The trick, then, is to apply a color to the reverse version of the lion mark that is lighter than the background it sits on!

See? clever!
