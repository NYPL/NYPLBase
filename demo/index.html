Elements of the page

Typography

NYPL's default typography is set to 16px, with a 24px line height. The heds and type size classes are set to the standard typographic scale (see Bringhurst).

Paragraphs are set to a maximum width of 60rem, to prevent an unreadably long line.

NYPL's core web typeface is Lato.

This is my substitute for pistol and ball.

With a philosophical flourish Cato throws himself upon his sword; I quietly take to the ship.

There is nothing surprising in this. If they but knew it, almost all men in their degree, some time or other, cherish very nearly the same feelings towards the ocean with me.

Calls-to-action

<button> tags are styled by default; other elements used as buttons get the .btn class applied.

NYPL buttons should also have the .gradient class applied to them. 

				<button class="gradient">Get this thing!</button>

Inactive buttons get the .btn--inactive class.

				<button class="btn--inactive">Get this thing!</button>

Sizes

You can add font classes too for bigger/smaller text.

				<button class="btn--small[[or large, or huge]]">Get this thing!</button>

Color variations

				<button class="btn--greenNeon">Get this thing!</button>

Specific NYPL use contexts

The Donate call-to-action is proved to work using this color/type design.

				<button class="btn--donate">Get this thing!</button>

Possible Friends-specific button style.

				<button class="btn--friends">Get this thing!</button>

Icon font

This is the current set of supported icons. We use a custom export of Icomoon.

Note to self: watch for width conflicts on elements with both a width class ("one-fifth") and an icon class or data-icon on the same element; icons have a width and height: auto setting.

How to use the icons

Class name

This is some text

				<p class="icon-cog">This is some text</p>

Data-icon attribute

This is also some text

				<p aria-hidden="true" data-icon="&#xf013;">This is also some text</p>

Mixin

Feel free to incorporate icons into your own styles, like this:

				.custom-zoom-out:before { @include nypl-icon("-"); }
				.custom-zoom-in:before { @include nypl-icon("+"); }

The logo

This is the logo (without the "New York Public Library" text, it's referred to as the "lion mark"), conveniently available as part of our icon font! Because it's an icon font, changing the color of the lion mark is as easy as applying a text color.

See? All I did was change the color of the text of the element with the icon. Now it's blue!

We also have a version of the lion mark in reverse. Here it is, no styling applied.

But what you see here is TOTALLY WRONG. Bad, wrong, no-no. Never use the lion mark with black/dark fills. The thing to remember is:
the fill is always lighter than the background!

The trick, then, is to apply a color to the reverse version of the lion mark that is lighter than the background it sits on!

See? clever!
